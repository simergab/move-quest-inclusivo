# Move Quest Inclusivo

Jogo web que usa a camera do dispositivo para reconhecer movimentos corporais com MediaPipe Pose.

## Como jogar

1. Abra o site em um navegador com permissao de camera.
2. Escolha um modo: Corrida virtual, Jogo de danca, Desafios ou Adaptado.
3. Pressione **Iniciar camera**.
4. Faca os movimentos mostrados na tela para ganhar pontos e moedas.
5. Use moedas para desbloquear roupas, habilidades e desafios para o avatar.

## Inclusao

O modo adaptado usa movimentos mais simples, ritmo mais lento, instrucoes com audio e feedback positivo constante.

## Deploy no Render

Este projeto e um site estatico. No Render, use:

- Build Command: vazio
- Publish Directory: `.`
